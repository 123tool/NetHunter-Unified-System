## KaliNetHunter Unified System

Fondasi infrastruktur digital yang dirancang untuk menyatukan berbagai lingkungan kerja (Termux, Linux, WSL) ke dalam satu ekosistem yang konsisten. Sistem ini secara otomatis mendeteksi hak akses perangkat dan menyiapkan alat pentesting, anonimitas, serta pengembangan SaaS dalam satu kali eksekusi.

## 🚀 Fitur Unggulan
- 🛡️ **Dual-Mode Intelligence:** Deteksi otomatis mode Root dan Non-Root dengan penyesuaian izin akses sistem secara *real-time*.
- 🧅 **Anonymity Layer:** Integrasi otomatis dengan jaringan Tor (SOCKS5) untuk mendukung alat seperti GhostMailer.
- 🐉 **NetHunter Deployment:** Modul opsional untuk menginstal lingkungan Kali NetHunter Rootless secara instan.
